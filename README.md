# jira-integration

Creation of a Jira issue through a React application.
Using React + Vite, Node.js + Express, and Jira APIv3.

## Project setup

- Node version: v20.18.0
- NPM version: 10.8.2

```bash
cd server
npm install
npm start
cd ../client
npm install
npm run dev
```
